Qt正则表达式

2022/10/26 C++ Qt 正则表达式

Qt中的正则表达式，主要是靠如下三个类来实现

  • QRegularExpression提供了一个可以用于匹配的正则表达式
  • QRegularExpressionMatch提供了一个匹配字符串的结果
  • QRegularExpressionMatchIterator提供了一个用于迭代结果的迭代器

匹配

我们可以在构造的时候设定模式串和一些选项，例如是否大小写敏感，.是否匹配换行符等等。

我们可以使用match方法来对一个QString进行匹配。注意，match的匹配仅仅匹配一次。遇到符合的串就不再匹配了。区分它和globalMath。

#include <QRegularExpression>
int main(int argc, char *argv[])
{
    QRegularExpression reg(R"(\bmov\b)");
    QString str("mov rax, rbp");
    QRegularExpressionMatch matched = reg.match(str);
    return 0;
}

可以使用hasMathed来判断正则表达式是否被匹配。

    QRegularExpressionMatch matched = reg.match(str);
    qDebug() << matched.hasMatch();

可以得到一个QRegularExpressionMatch。那我们如何获得具体的结果呢？

捕获组

我们将用()扩起来的表达式称为一个捕获组。就像这样(hello|world)。

又或者是多个捕获组的组合(hello|world)(.)?。在这个表达式里有两个捕获组，一个是(hello|world)，一个是(.)。

我们会对捕获组进行编号，在展开讨论之前，需要说明的是，编号为0的捕获组，指的是正则表达式整体，这一规则在支持捕获组的语言中，基本上都是适用的。我们可以很轻松的验证这一点。

对于捕获组的编号， 是从左向右的。比如(\d{4})-(\d{2})-(\d{2})。的四个捕获组

  1. \d{4})-(\d{2})-(\d{2})
  2. (\d{4})
  3. (\d{2})
  4. (\d{2})

如果包含捕获组的嵌套关系，依旧是这样。例如(\d{4})-(\d{2}-(\d\d))。则

编号 命名 捕获组 匹配内容
0 (\d{4})-(\d{2}-(\d\d)) 2008-12-31
1 (\d{4}) 2008
2 (\d{2}-(\d\d)) 12-31
3 (\d\d) 31

在Qt中，我们可以使用captured 这个方法来返回捕获组捕获的 （匹配的）字符串。

    QRegularExpression reg(R"((\d{4})-(\d{2})-(\d{2}))");
    QString str = "2022-10-26";
    QRegularExpressionMatch matched = reg.match(str);
    qDebug() << matched.hasMatch();
    qDebug() << matched.captured(0);
    qDebug() << matched.captured(1);
    qDebug() << matched.captured(2);
    qDebug() << matched.captured(3);

当然，每个捕获组都有匹配的内容。当这个正则表达式被匹配后，每个捕获组的匹配的内容是唯一的。

Qt中可以使用captured来获得捕获组捕获的内容。

#include <QRegularExpression>
#include <QDebug>
int main(int argc, char *argv[])
{
    QRegularExpression reg(R"((\d{4})-(\d{2})-(\d{2}))");
    QString str = "Today is 2022-10-26";
    QRegularExpressionMatch matched = reg.match(str);
    qDebug() << matched.captured(0);
    qDebug() << matched.captured(1);
    qDebug() << matched.captured(2);
    qDebug() << matched.captured(3);
    return 0;
}

你也可以用

  • capturedLength(int nth=0) 获得捕获的字符串的长度

  • capturedStart(int nth=0) 获得捕获的字符串相起始位置对于整个字符的下标

  • capturedEnd(nth=0) 获得捕获的字符串相结束位置对于整个字符的下表。

等函数来获取捕获的相关信息，其中nth为第n个捕获组。

详情请参阅：Q正则表达式匹配类 |Qt 酷睿 6.4.0

全局匹配

使用match匹配，是非全局的。举一个例子。I say hello and you say hello too 匹配hello。

  • math -> I say hello and you say hello too
  • globalMath -> I say *hello* and you say *hello* too

可以看见，全局匹配会匹配所有符合条件按的字串。

获取全局匹配的结果

QRegularExpressionMatchIterator是一个JAVA风格的迭代器，用于迭代匹配后的结果。

使用next方法获得QRegularExpressionMatch，我们便可以进行其他操作。

#include <QRegularExpression>
#include <QDebug>
int main(int argc, char *argv[])
{
    QRegularExpression reg(R"(hello\w?)");
    QString s("I say hello and you say helloo too");
    QRegularExpressionMatchIterator it = reg.globalMatch(s);
    int i = 0;
    while(it.hasNext()) {
        ++i;
        auto tmp = it.next();
        qDebug() << QString("第%1个匹配结果: ").arg(i) << Qt::endl
                 << tmp.captured(0) << Qt::endl
                 << QString("len: %1").arg(tmp.capturedLength()) << Qt::endl
                 << QString("start index: %1").arg(tmp.capturedStart());
    }
    return 0;
}
